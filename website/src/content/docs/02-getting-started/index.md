# Getting Started

Kitledger projects are standard TypeScript applications. [cite_start]You combine core packages with your own code to define behavior[cite: 75, 83].

## Prerequisites

* Node.js (LTS)
* [cite_start]PostgreSQL Database [cite: 104]
* TypeScript

## Quick Summary

1.  [cite_start]**Scaffold:** Create a new TypeScript project[cite: 75].
2.  [cite_start]**Define:** Create Transaction, Entity, and Unit models using `@kitledger/core`[cite: 98].
3.  [cite_start]**Configure:** Connect the database and Admin UI[cite: 101, 103].
4.  [cite_start]**Run:** Start the server and access your generated system[cite: 114].

Proceed to the Quickstart guide to run your first single-file server.
